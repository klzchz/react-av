# Aplicativo Razzie


## Instalação
1. Clone o repositório:
    ```bash
    git clone https://github.com/seunomeusuario/razzie-app.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd razzie-app
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```

## Uso
Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
